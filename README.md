# Filther - your best array duplicates extinguisher

Filtering Arrays for duplicates

Use it like this

```js 
import f from 'filther';

let arr = [23, 23, 23, 23, 23, 45, 45, 45, 567, 67, 67, 67, 67, 67];

let filtered = f(arr);

console.log(filtered) // should return [23,45,567,67] 
```
